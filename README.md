# Student Database Application

This project is a simple C# Console Application built using .NET Framework and Entity Framework 6 Code First approach.

## Features

- Create and store student records
- Entity Framework 6 integration
- Code First workflow
- SQL Server database support
- Basic CRUD database operations

## Technologies Used

- C#
- .NET Framework
- Entity Framework 6
- SQL Server
- Visual Studio

## Project Structure

- Student.cs - Student model class
- StudentContext.cs - Database context class
- Program.cs - Application entry point

## Example

This application creates a student record and saves it to the database using Entity Framework.
